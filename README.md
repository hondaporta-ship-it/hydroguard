# HydroGuard v0.3 アップデート

## 🆕 v0.3の変更点

1. **配色刷新** — 黒×真っ赤 → アイボリー × オーシャンブルー
   - 警備員の心理的負担を軽減（毎日見る道具なので）
   - 「Hydro=水分」の意味と視覚イメージが一致
   - 警告色（赤）は危険時のアラートにのみ使用

2. **緊急発信ボタン3つ実装**
   - 🌊 **管制に緊急連絡** — 1タップで本田携帯（070-4734-1911）に発信画面 ※テスト期間ダミー
   - 🚓 **警察 110** — 2タップ確認式（誤発報防止）
   - 🚒 **消防 119** — 2タップ確認式（誤発報防止）

3. **取消ボタン追加**
   - 「↶ 直前を取消」で誤タップを1回戻せる
   - 既存の「リセット」（隊長のみ）はそのまま継続

4. **アイコン更新**
   - アイボリー背景 + 紺×オーシャンの「HG」

## 🚀 OpenClawへの依頼方法

既存の `~/Documents/hydroguard-deploy` フォルダ（または前回作業したパス）に
このフォルダのファイルを**上書きコピー**してから、deploy.shを再実行するだけでOKです。

### OpenClawに渡す指示文（コピペ用）

```
HydroGuard v0.3 のアップデート版です。
~/Downloads/hydroguard-v3/ に新しいビルド済みファイル一式があります。

【作業内容】
1. 既存の hydroguard プロジェクトディレクトリ（前回 ./deploy.sh を実行した場所）を確認
2. 以下のファイルを新版で上書き：
   - bundle.js
   - styles.css
   - index.html
   - manifest.json
   - icon-192.png
   - icon-512.png
3. ./deploy.sh を再実行してGitHubにpush
4. 数分後、https://hondaporta-ship-it.github.io/hydroguard/ に反映されているか確認

【主な変更点】
- 配色をアイボリー×オーシャンブルーに刷新
- 緊急連絡先 tel:リンク 実装（管制=07047341911、警察110、消防119）
- 警察消防は誤発報防止の2タップ確認式モーダル
- 直前の記録を取り消す「↶ 取消」ボタン追加

外部送信や個人情報アップロードは一切なし。
deploy.shの中身は前回確認済み。同じスクリプトを叩くだけなので
内容を再確認する必要はないが、念のため diff で確認してから実行してOK。
```

## ⚠️ 既知の制約

- **緊急電話のtel:リンクはiPhone必須** — Mac/iPad（電話なし）では動作しない（ダイヤラ画面が出るだけ）
- **管制連絡先はテスト期間中ダミー** — ヒデさんの携帯070-4734-1911に向かう。テスト後に樫迫さん携帯 or ATS代表番号に切り替え予定
- **localStorage制限** — Safariのプライベートブラウジングモードでは保存できない（隊員には通常モードでホーム画面追加してもらう）
